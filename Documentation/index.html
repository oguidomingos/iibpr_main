Installation

Before actually putting the template to use, there are certain installation steps to be followed, which are as follows:
We insist you to download Template to edit website template.

  1. After file download, you will require it to unzip in case to use the files.
  2. You can use code editor like Sublime Text, Notepad++, Atom, etc. for editing template source code files.
  3. For uploading template files to server you will also require FTP manager. You can use file manager at your hosting control panel or application such as FileZilla, CuteFTP, etc.
  4. As we used Pure HTML,CSS & JS/jQuery you just need to put template's root folder to your server.

Hosting

To start using the template correctly, entire files in the template folder has to be unzipped later needs to be uploaded to hosting server.
Follow below steps to get started with your Site Template:

  1. Open the ... /Technetium-template folder to find all the Templates Files
  2. You will need to Upload these files & folders to your Web Server using FTP.
  3. Upload the required files/folders as listed below:
    • /css - Stylesheets Folder
    • /fonts - Fonts Folder
    • /images - Images Folder
    • /js - Javacripts Folder
    • /index.html - Index File/Homepage
    • /home2.html - Second Index File/Homepage
    • /home3.html - Third Index File/Homepage
    • /home4.html - Fourth Index File/Homepage
    • /home5.html - Fifth Index File/Homepage
    • /home6.html - Sixth Index File/Homepage
    • /home7.html - Seventh Index File/Homepage
    • /about-us.html - About Page
    • /partners.html - Team Listing / Partners Page
    • /partners-details.html - Team Details / Partners Details Page
    • /blogs.html - Blogs Listing Page
    • /blog-details.html - Blog Details Page
    • /contact.html - Contact Page
    • /pricing.html - Pricing Page
    • /services.html - Services Listing Page
    • /services-details.html - Services Details Page
    • /portfolio-listing.html - Portfolio Listing Page
    • /portfolio-Details.html - Portfolio Details Page
    • /faq.html - FAQ Page
    • /404.html - 404 / Error Page
    • /coming-soon.html - Coming Soon Page
    • /ajax.php - Required for Contact Form, Subscribe Form and Blog Comment Form
    The other files can be used according to your utility and purpose.
  4. After adding all these folders, now you are free to edit template to draw in more and more customers.

Basic template Structure

The template has a responsive layout and is based on the Bootstrap V4.3.1 Framework. Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web. Click Here to know more about Bootstrap.

HTML Structure

The template follows a common HTML structure. Below is the common structure followed.

                  <!doctype html>
<html lang="en">
<head>
	[Page's meta, title, stylesheets etc...]
</head>
<body>
	[LOADER]
	
	<!----- Header Start ----->
	<header>
		[LOGO]
		[MOBILE NAVIGATION BUTTON]    
		[NAVIGATION]
	</header> 
	<!----- Header End ----->

	[MAIN CONTENT GOES HERE] 

	<!----- Footer Start ----->

	[FOOTER_CONTENT] 

	<!----- Footer End ----->

	[PAGE JAVASCRIPTS HERE]

</body>
</html>
